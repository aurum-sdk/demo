# Aurum Demo

Playground for `@aurum-sdk/core`

## Customization Options

### Connect UX

| Option | Description                                                      |
| ------ | ---------------------------------------------------------------- |
| Mode   | Widget (embedded), Modal (popup dialog), or Headless (custom UI) |

### Branding

| Option         | Description                                  |
| -------------- | -------------------------------------------- |
| Theme          | Toggle between dark and light mode           |
| Border Radius  | Choose from none, sm, md, lg, xl             |
| Brand Color    | Set the accent color via picker or hex input |
| Logo URL       | Add a custom logo (URL or data URI)          |
| Font           | Select from available font options           |
| Aurum Branding | Show or hide the "Powered by Aurum" footer   |

### Wallets

| Option          | Description                                     |
| --------------- | ----------------------------------------------- |
| Layout          | Stacked or Grid layout for wallet list          |
| Enabled Wallets | Toggle which wallets should appear in the modal |

## Built With

- React + TypeScript
- Vite
- Tailwind
- shadcn/ui components
- `@aurum-sdk/core`
